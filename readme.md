# موقع ذرة سكر - Dharat Sokar Website

موقع ويب احترافي وفاخر لعلامة "ذرة سكر" - حلويات منزلية في دبي

## المميزات

✅ تصميم فاخر بألوان ناعمة (وردي فاتح، بيج، أبيض)
✅ دعم عربي RTL كامل من اليمين إلى اليسار
✅ متجاوب مع جميع أحجام الشاشات (Desktop, Tablet, Mobile)
✅ تكامل واتساب للطلب المباشر
✅ صور عالية الجودة من CDN
✅ تأثيرات احترافية وسلسة

## الأقسام

1. **القسم الرئيسي (Hero)** - صورة فاخرة مع عنوان وأزرار
2. **من نحن** - وصف العلامة التجارية
3. **لماذا نحن** - 4 مميزات رئيسية
4. **المنتجات** - عرض 4 منتجات مع الأسعار
5. **معرض الصور** - 4 صور فاخرة
6. **قسم الطلب** - زر واتساب بارز
7. **التواصل** - معلومات التواصل والموقع

## كيفية النشر

### الطريقة 1: Netlify (الأسهل)
1. اذهب إلى https://netlify.com
2. اختر "Drag and drop your site folder"
3. اسحب مجلد `dharat-sokar-website`
4. سيحصل موقعك على رابط دائم مجاني

### الطريقة 2: Vercel
1. اذهب إلى https://vercel.com
2. اختر "New Project"
3. اختر المجلد `dharat-sokar-website`
4. انقر "Deploy"

### الطريقة 3: GitHub Pages
1. أنشئ مستودع GitHub
2. ارفع الملفات
3. فعّل GitHub Pages من الإعدادات
4. سيحصل موقعك على رابط مجاني

## التخصيص

### تغيير الألوان
عدّل متغيرات CSS في `styles.css`:
```css
:root {
    --primary-color: #E8B4C8;      /* اللون الأساسي (وردي) */
    --secondary-color: #F5E6D3;    /* اللون الثانوي (بيج) */
    --accent-color: #8B6F47;       /* لون التأكيد (بني) */
}
```

### تحديث المنتجات
عدّل قسم المنتجات في `index.html` بإضافة/حذف بطاقات المنتجات

### تحديث الصور
استبدل روابط الصور من CDN برابط صورك الخاصة
